# Sistema de Ventas - SQL Server

Proyecto de base de datos desarrollado en SQL Server.

## Descripción
Sistema de ventas que permite:
- Registrar clientes
- Registrar productos
- Registrar ventas
- Consultar ventas y totales vendidos

## Contiene
- Creación de base de datos
- Tablas con llaves primarias y foráneas
- Inserción de datos de prueba
- Registro de ventas
- Consultas con JOIN y SUM

## Tecnologías
- SQL Server
- SQL Server Management Studio
- Visual Studio Code
- Git y GitHub
